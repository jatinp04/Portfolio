Project Overview

Our project is a web application that displays GPS device data on a map, including the device location, latest timestamp, device type, and device ID. The application is built using the PERN stack, which includes PostgreSQL, Express.js, React, and Node.js. JWT authentication is used to secure the user login and REST APIs are used to handle data requests and responses.
